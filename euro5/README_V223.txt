Easy Gym v223

This is the complete GitHub upload package for Easy Gym v223.

LOG DOWNLOADS
- Word and PDF are A4 portrait.
- Max Ever and Max Today use the original coloured icons with sufficient vertical space.
- Existing LOG layout and app interface are retained.

MODULAR LANGUAGE PACKAGES
- Each interface language is an independent file in /lang.
- English, German, French, Italian, and Spanish are loaded directly and reliably.
- Edit /lang/config.js to enable, disable, reorder, or change the default language.
- Edit one language file without touching the other languages.
- Copy /lang/_template.js to add another language; loader.js loads additional enabled packages.
- Detailed instructions: /lang/README_LANGUAGE_PACKS.txt
